# Text

代码：

```swift
Text("Tap me !") 
    .font(.largeTitle)
    .fontWeight(.bold)
    .foregroundColor(.white)
    .padding(30)
    .frame(alignment: .center)
    .background(Color.red)
    .cornerRadius(100)
```

可以设置的样式：

* text
* font color
* font size
* font weight
* spacing of the words
* size of the Text
* background color
* corner radius
